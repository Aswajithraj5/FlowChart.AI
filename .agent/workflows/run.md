---
description: how to run the FlowChart.AI application
---

To run the application locally, follow these steps:

1. Open a terminal in the project root directory (`c:\Users\aswaj\OneDrive\Desktop\FlowChart.AI`).
2. Run the following command:
```bash
npm run dev
```
3. Once the server starts, open your browser and navigate to:
[http://localhost:5173/](http://localhost:5173/)
